 # Netflix-Data-Analysis-using-SQL-Python

## Project Objectives
To integrate Python with MySQL for efficient data analysis and database management.

To perform data cleaning and preprocessing on the movie dataset.

To analyze movie trends based on popularity, ratings, genres, languages, and release years.

To identify the most popular movies and highest-rated genres.

To create visualizations that provide meaningful business and entertainment industry insights.

To demonstrate an end-to-end Data Analytics workflow using SQL and Python.

## Analysis Performed
Created and configured a MySQL database for movie data storage.

Imported and cleaned the dataset by identifying missing values and duplicates.

Performed SQL-based exploratory data analysis using aggregate functions and grouping operations.

Identified the Top 10 Most Popular Movies.

Analyzed average ratings across different genres.

Created aggregated summary tables for genre-wise analysis.

Examined movie distribution based on genres, languages, and release years.

Connected Python with MySQL to retrieve and analyze data.

Generated visualizations to explore movie popularity, ratings, and trend patterns.

## Key Insights
Certain movie genres consistently achieved higher average ratings than others.

A small number of movies contributed significantly to overall popularity scores.

Genre distribution analysis revealed the most dominant movie categories.

Movie production showed noticeable growth trends over the years.

Popularity and ratings demonstrated varying relationships across genres.
